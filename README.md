## âœ¨ Features

- Live overlay showing coverage of 128Ã—128 map tiles  
- Green cells = mapped tiles  
- Red cells = unmapped tiles  
- Auto-map mode (automatically creates and checks maps)  
- Automatically moves scanned maps into your main inventory  
- HUD coverage percentage display  
- Red HUD warning when inventory is full  
- Configurable overlay size & placement  
- Clickable cells (jump focus / inspect area)  
- Lightweight & client-side only
